Set Up Instructions

Before using this stack please make sure you have followings installed:

  • MongoDB
  • NodeJS
  • nodemon

Run npm install to install dependencies.

Add MongoDB installation binaries path into environment variable. To start the app:

Start mongod in separatec cmd propmt window. This willl start mongodb instance. Then Run nodemon in separate window.

By default server will run on http://localhost:3000.

Example Usage

Inside your module file import your database file

var db = require("../../database/database.js");

And use your database object inside route like:

db.User.find({email: req.body.email, password: req.body.password}, function (err, results) {
	// do something with results        
});
